# Syslog.Server

A simple .NET Core multi threaded syslog server.

## PreRequisite 

### Build
- Visual Studio
- DotNet Core 2.0
- Open UDP Port 514 (syslog)

### Run
- DotNet Core 2.0
- Open UDP Port 514 (syslog)

## How to build
- Clone the Repo
- Use the Visual Studio Publish function (right click on the project -> publish)

## How to run
- dotnet run [Name of the compiled dll] [syslog.log path]
